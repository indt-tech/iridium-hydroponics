# Layout component

By default Protofy autogenerated pages from admin panel include a default layout. This layout is called `DefaultLayout` and can be 
customized by editting the following props:
<br/>
<table>
  <tbody>
    {[
      { propName: 'Prop', description: 'Description', type: 'Type' },
      { propName: 'children', description: 'Inner react components', type: 'React.ReactNode' },
      { propName: 'footer', description: 'Footer of the page', type: 'React.ReactNode' },
      { propName: 'header', description: 'Header of the page', type: 'React.ReactNode' },
      { propName: 'sideMenu', description: 'Workspace side menu', type: 'React.ReactNode' },
      { propName: 'themeSwitcher', description: 'Enables theme switch button', type: 'boolean' },
      { propName: 'tintSwitcher', description: 'Enables tint switch', type: 'boolean' },
    ].map(item => (
      <tr key={item.propName}>
        <td>{item.propName}</td>
        <td>{item.type}</td>
        <td>{item.description}</td>
      </tr>
    ))}
  </tbody>
</table>
