# Prerequisites

You have to clone the repository using `git`. 

Check that your sistem has already installed git by executing: `git --version`

## Clone the project
Clone the project by running:
```bash
git clone https://github.com/Protofy-xyz/Protofy.git
cd Protofy
```
## Run the project
Protofy can be run using local environment or Docker.
